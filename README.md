# Weather Forecast Android App

A modern Android weather application developed in Java using Android Studio.

The application allows users to search weather information by city name and displays real-time weather data using the OpenWeatherMap API.

---

## Features

- Search weather by city name
- Real-time weather data
- Temperature display
- Weather description
- Humidity and wind speed
- Weather icons
- Error handling
- Internet connection checking
- MVC architecture
- Modern UI design

---

## Technologies Used

- Java
- Android Studio
- OpenWeatherMap API
- ConstraintLayout
- AsyncTask
- MVC Architecture
